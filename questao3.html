Clculo dos Parmetros de Projeto

Para um sistema de segunda ordem, as relaes fundamentais so:

  • d = /tp (frequncia amortecida a partir do tempo de pico)
  • n = d/(1-) (frequncia natural no amortecida)
  • = n (parte real dos plos dominantes)
In[23]:
# Especificaes do projeto
tp_desired = 1.047  # Tempo de pico desejado (s)
zeta_pid = 0.8      # Coeficiente de amortecimento

# Clculo da frequncia amortecida
wd_pid = np.pi / tp_desired
print(f"Frequncia amortecida: d = /tp = /{tp_desired} = {wd_pid:.3f} rad/s")

# Clculo da frequncia natural
wn_pid = wd_pid / np.sqrt(1 - zeta_pid**2)
print(f"Frequncia natural: n = d/(1-) = {wd_pid:.3f}/(1-{zeta_pid}) = {wn_pid:.3f} rad/s")

# Parte real dos plos dominantes
sigma_pid = zeta_pid * wn_pid
print(f"Parte real dos plos:  = n = {zeta_pid}  {wn_pid:.3f} = {sigma_pid:.3f}")

# Plos dominantes desejados
s_dom = -sigma_pid + 1j*wd_pid
print(f"Plos dominantes desejados: s1,2 = {-sigma_pid:.3f}  j{wd_pid:.3f}")

Frequncia natural: n = d/(1-) = 3.001/(1-0.8) = 5.001 rad/s

Projeto do Controlador PID

Estrutura do Controlador PID

O controlador PID tem a forma: Gc(s) = Kp + Ki/s + Kds = (Kds + Kps + Ki)/s

Mtodo de Alocao de Plos

Para garantir que os plos dominantes estejam na posio desejada, utilizaremos o mtodo de alocao de plos. O sistema em malha fechada ter 3 plos (sistema de 3 ordem devido ao integrador do PID).

In[25]:
# Para um sistema de terceira ordem, precisamos de 3 plos
# 2 plos dominantes (complexos conjugados) + 1 plo adicional (real)

# O plo adicional deve estar suficientemente longe dos plos dominantes
# para no interferir significativamente na resposta
p_additional = 10 * sigma_pid  # Plo adicional 10 vezes mais rpido
print(f"Plo adicional escolhido: -{p_additional:.3f}")

# Polinmio caracterstico desejado
# (s + 2ns + n)(s + p_additional)
desired_poles = [-sigma_pid + 1j*wd_pid, -sigma_pid - 1j*wd_pid, -p_additional]
desired_char = np.poly(desired_poles)

print(f"Polinmio caracterstico desejado:")
print(f"s + {desired_char[1]:.3f}s + {desired_char[2]:.3f}s + {desired_char[3]:.3f}")

Clculo dos Parmetros do PID

In[26]:
# Sistema com PID em malha fechada:
# T(s) = Gc(s)G(s) / (1 + Gc(s)G(s))
# 
# Onde: Gc(s) = (Kds + Kps + Ki)/s
#       G(s) = K/[(s+1)(s+4)]
#
# Funo de transferncia em malha aberta:
# Gc(s)G(s) = K(Kds + Kps + Ki) / [s(s+1)(s+4)]
#
# Denominador da malha fechada:
# s(s+1)(s+4) + K(Kds + Kps + Ki) = 0
# s + 5s + 4s + KKds + KKps + KKi = 0
# s + (5 + KKd)s + (4 + KKp)s + KKi = 0

print("Igualando coeficientes do polinmio caracterstico:")
print("Coeficiente de s: 1 = 1 ")
print(f"Coeficiente de s: {desired_char[1]:.3f} = 5 + KKd")
print(f"Coeficiente de s: {desired_char[2]:.3f} = 4 + KKp") 
print(f"Coeficiente de s: {desired_char[3]:.3f} = KKi")

# Calculando os parmetros
Kd_pid = (desired_char[1] - 5) / K3
Kp_pid = (desired_char[2] - 4) / K3
Ki_pid = desired_char[3] / K3

print(f"\nParmetros do controlador PID:")
print(f"Kd = ({desired_char[1]:.3f} - 5)/{K3} = {Kd_pid:.3f}")
print(f"Kp = ({desired_char[2]:.3f} - 4)/{K3} = {Kp_pid:.3f}")
print(f"Ki = {desired_char[3]:.3f}/{K3} = {Ki_pid:.3f}")

print(f"\nControlador PID final:")
print(f"Gc(s) = {Kp_pid:.3f} + {Ki_pid:.3f}/s + {Kd_pid:.3f}s")

Igualando coeficientes do polinmio caracterstico:
Coeficiente de s: 1 = 1 
Coeficiente de s: 48.009 = 5 + KKd
Coeficiente de s: 345.130 = 4 + KKp
Coeficiente de s: 1000.566 = KKi

Parmetros do controlador PID:
Kd = (48.009 - 5)/5 = 8.602
Kp = (345.130 - 4)/5 = 68.226
Ki = 1000.566/5 = 200.113

Controlador PID final:
Gc(s) = 68.226 + 200.113/s + 8.602s

# Funo para anlise da resposta ao degrau
def analyze_step_response(t, y):
    """Analisa caractersticas da resposta ao degrau"""
    yss = y[-1]  # Valor final
    
    # Tempo de pico
    peak_idx = np.argmax(y)
    tp = t[peak_idx]
    peak_value = y[peak_idx]
    
    # Sobressinal percentual
    Mp = (peak_value - yss) / yss * 100
    
    # Tempo de acomodao (critrio de 2%)
    settling_band = 0.02 * yss
    settled_idx = np.where(np.abs(y - yss) <= settling_band)[0]
    ts = t[settled_idx[0]] if len(settled_idx) > 0 else t[-1]
    
    # Tempo de subida (10% a 90%)
    y10, y90 = 0.1 * yss, 0.9 * yss
    idx10 = np.where(y >= y10)[0][0] if np.any(y >= y10) else 0
    idx90 = np.where(y >= y90)[0][0] if np.any(y >= y90) else len(y)-1
    tr = t[idx90] - t[idx10]
    
    return {
        'Valor Final': yss,
        'Tempo de Pico (s)': tp,
        'Sobressinal (%)': Mp,
        'Tempo de Acomodao (s)': ts,
        'Tempo de Subida (s)': tr
    }
